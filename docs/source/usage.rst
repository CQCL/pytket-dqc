Installation
------------

To use pytket-dqc requires the instillation of
:ref:`kahypar <https://github.com/kahypar/kahypar>` and 
:ref:`graphviz <https://graphviz.org/download/>`. pytket-dqc can then be 
installed using pip:

.. code-block:: console

   > pip install .
